# E-commerce Site

This project is a basic e-commerce website built with HTML, CSS, JavaScript, PHP, and MySQL. It is designed to run locally on a XAMPP server, making it easy to set up and customize for personal or learning use.

## Features
- Displays a list of products fetched from a MySQL database.
- Allows users to add products to a shopping cart.
- Simple, responsive design for an optimal experience on various devices.

## Technologies Used
- HTML, CSS, JavaScript
- PHP
- MySQL
- XAMPP (Apache and MySQL server for local hosting)

## Getting Started

Follow these steps to set up the project locally:

### Prerequisites
1. **XAMPP**: Download and install XAMPP from [Apache Friends](https://www.apachefriends.org/index.html) if not already installed. XAMPP includes Apache and MySQL, which are required for running this application locally.

2. **Git**: Install Git if you haven't already. [Git Download](https://git-scm.com/).
